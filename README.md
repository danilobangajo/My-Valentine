## How to Use

1. Open `index.html` in your web browser
2. Click on the envelope to open it
3. Answer the Valentine's question
4. If "Yes" - enjoy the love letter and photo album
5. If "No" - see the broken heart animation (with retry option)

## Customization

### Change the Letter Content
Edit the text in `index.html` between the `<div class="letter-body">` tags (lines 67-77)

### Add Your Photos
1. Place your 2 photos in the same folder as the HTML file
2. Name them `photo1.jpg` and `photo2.jpg` (or update the filenames in HTML)
3. Recommended size: 800x600px or similar aspect ratio

### Change Photo Captions
Edit the text in `<div class="photo-caption">` tags in `index.html` (lines 88 and 94)

### Customize Colors
Edit `styles.css`:
- Main gradient: Line 9 (body background)
- Envelope color: Line 56 (flap color)
- Button colors: Lines 165-167 (btn-yes) and 175-177 (btn-no)

### Change Names
- Replace "Anna Liza" with your girlfriend's name in `index.html` (line 66)
- Replace "Danilo" with your name in `index.html` (line 79)
